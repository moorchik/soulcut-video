# Відео для портфоліо

Покладіть сюди файли відео (.mp4). Вони будуть доступні на сайті за адресою `/videos/назва-файлу.mp4`.

**Приклад:** якщо файл називається `Highlight_Kiera-and-Sascha.mp4`, у HTML вказуйте:
`videos/Highlight_Kiera-and-Sascha.mp4`

**Обмеження GitHub:** один файл — до 100 МБ. Для більших відео використовуйте Vimeo або інший хостинг.
