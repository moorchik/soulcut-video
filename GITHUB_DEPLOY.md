# Як викласти сайт на безкоштовний хостинг GitHub Pages

Цей сайт можна опублікувати в інтернеті безкоштовно через **GitHub Pages**. Нижче — покрокова інструкція.

---

## Крок 1. Встановити Git (обовʼязково, якщо команда `git` не працює)

1. Завантажте установник для Windows: **https://git-scm.com/download/win**  
2. Запустіть завантажений файл і встановіть Git (достатньо стандартних налаштувань — просто натискайте Next).  
3. **Важливо:** закрийте термінал у Cursor і відкрийте його знову (або перезапустіть Cursor), щоб система підхопила команду `git`.  
4. Перевірте: у терміналі введіть `git --version`. Має зʼявитися щось на кшталт `git version 2.x.x`.

---

## Крок 2. Створити репозиторій на GitHub

1. Увійдіть на https://github.com у свій акаунт.  
2. Натисніть зелений **New** (або **Create repository**).  
3. Заповніть:
   - **Repository name:** наприклад `soulcut-video` (або будь-яка назва латиницею).
   - **Public.**  
   - **НЕ** ставлячи галочки "Add a README" / "Add .gitignore" — репозиторій має бути **порожнім**.  
4. Натисніть **Create repository**.  
5. Скопіюйте URL репозиторія — він виглядатиме так:  
   `https://github.com/ВАШ_ЛОГІН/soulcut-video.git`

---

## Крок 3. Завантажити проєкт у репозиторій

Відкрийте **термінал** (Terminal) у Cursor у папці проєкту `SOULCUT VIDEO` і виконайте команди по черзі. Замініть `ВАШ_ЛОГІН` та `soulcut-video` на свої значення.

```bash
git init
git add .
git commit -m "Initial commit: Soulcut Video website"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГІН/soulcut-video.git
git push -u origin main
```

При першому `git push` можуть попросити увійти в GitHub (через браузер або логін/пароль). Якщо у вас увімкнена **двофакторна автентифікація**, замість пароля потрібно використати **Personal Access Token**:  
GitHub → Settings → Developer settings → Personal access tokens → Generate new token (з правами `repo`).

---

## Крок 4. Увімкнути GitHub Pages

1. У репозиторії на GitHub відкрийте **Settings** (Налаштування).  
2. Зліва виберіть **Pages**.  
3. У блоці **Source** (Джерело) виберіть:
   - **Branch:** `main`  
   - **Folder:** `/ (root)`  
4. Натисніть **Save**.

Через 1–2 хвилини сайт буде доступний за адресою:

**https://ВАШ_ЛОГІН.github.io/soulcut-video/**

(замість `soulcut-video` — назва вашого репозиторія).

---

## Оновлення сайту пізніше

Після змін у файлах виконайте в папці проєкту:

```bash
git add .
git commit -m "Опис змін"
git push
```

Сайт на GitHub Pages оновиться автоматично за кілька хвилин.

---

**Мені не потрібен ваш пароль або токен.** Всі кроки ви робите самі у своєму браузері та терміналі; я лише підготував проєкт і цю інструкцію.
