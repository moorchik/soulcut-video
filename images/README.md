# Зображення для сайту

Покладіть сюди фото — пропорції для ПК і мобільних:

---

## Фон головного екрану (hero)

- **Файл:** `hero-bg.jpg` (або .png)
- **Пропорції:** **1920×1080** (16:9) або **2400×1350**
- Чим замінити: покладіть файл у цю папку — він автоматично з’явиться на фоні. Важливе (обличчя, пара) тримайте **по центру** — на телефонах краї будуть обрізані.

Якщо хочете тільки картинку без відео — у `index.html` додайте до тегу `<header class="hero">` клас `hero--image-only`, тобто:  
`<header class="hero hero--image-only" id="hero">`.

---
