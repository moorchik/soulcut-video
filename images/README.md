# Зображення для сайту

Покладіть сюди фото — пропорції для ПК і мобільних:

---

## Фон головного екрану (hero)

- **Файл:** `hero-bg.jpg` (або .png)
- **Пропорції:** **1920×1080** (16:9) або **2400×1350**
- Чим замінити: покладіть файл у цю папку — він автоматично з’явиться на фоні. Важливе (обличчя, пара) тримайте **по центру** — на телефонах краї будуть обрізані.

Якщо хочете тільки картинку без відео — у `index.html` додайте до тегу `<header class="hero">` клас `hero--image-only`, тобто:  
`<header class="hero hero--image-only" id="hero">`.

---

## Іконка сайту (favicon)

- **Файл:** `icon.svg` — відображається у вкладці браузера та при збереженні в закладки.

---

## Обкладинка при відправці посилання (Telegram, соцмережі)

- **Файл:** `banner.png` (або .jpg)
- **Рекомендований розмір:** **1200×630** пікселів — так краще виглядає в прев’ю (Telegram, Facebook, Twitter тощо).
- Покладіть файл у цю папку під назвою `banner.png`. Якщо використовуєте .jpg — у `index.html` замініть у мета-тегах `og:image` і `twitter:image` на `banner.jpg`.

---

## Банери портфоліо (highlights)

- **Пропорції:** **2500×1000** (2.5:1) — одна форма для всіх банерів.
- Приклади файлів: geoff-lauren.png, paige-mike.png, carlee-jevan.png.
